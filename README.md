# In-memmory File System (IFS)

## Requirements
- Python 2.7
- pip

## Installation
```
pip install -r requirements.txt
```

## File operations
- Create file
- Read file
- Update / append file
- Delete file

## Directory operations 
- Create Directory
- List Directory
- Delete Directory (recursively)
